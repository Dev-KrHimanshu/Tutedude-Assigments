# Assignment 1 - Python Basics

## Overview
This assignment focuses on basic Python programming concepts including mathematical operations and user input handling.

## Tasks

### Task 1: Basic Mathematical Operations
**Problem Statement:** Write a Python program that performs basic mathematical operations on two numbers.

**Features:**
- Takes two numbers as input from the user
- Performs the following operations:
  - Addition
  - Subtraction
  - Multiplication
  - Division
- Displays the results in a formatted manner

**How to Run:**
```bash
python Task1.py
```

**Example Output:**
```
Enter the first number: 10
Enter the second number: 5

Results:
Addition: 10.0 + 5.0 = 15.0
Subtraction: 10.0 - 5.0 = 5.0
Multiplication: 10.0 * 5.0 = 50.0
Division: 10.0 / 5.0 = 2.0
```

### Task 2: String Concatenation and Greeting
**Problem Statement:** Write a Python program that takes a user's first and last name as input and displays a personalized greeting message.

**Features:**
- Takes first name and last name as input from the user
- Concatenates the names into a full name
- Displays a personalized greeting message

**How to Run:**
```bash
python Task2.py
```

**Example Output:**
```
Enter your first name: Himanshu
Enter your last name: Singh
Hello, Himanshu Singh! Welcome to the Python Program!
```
